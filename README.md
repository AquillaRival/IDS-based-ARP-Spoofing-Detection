# 🚨 ARP Spoofing IDS Dashboard (Full-Stack Security Project)

## 📖 Overview

This is a full-stack Intrusion Detection System (IDS) designed to monitor a local network for ARP Spoofing attacks in real-time. It uses a Python-based network agent to detect suspicious MAC address changes and a Node.js/Express backend to log these events into a MySQL database. The alerts are displayed on a live, auto-refreshing web dashboard.

Built as a Computer Science Engineering semester mini-project to integrate networking, cybersecurity, and full-stack web development.

## 🚀 Features

- **Real-Time Network Monitoring:** Continuously scans the OS ARP table.
- **Intrusion Detection:** Instantly detects fake MAC addresses attempting to spoof the router or other devices.
- **RESTful API:** Python agent communicates securely with the Node.js backend via HTTP POST requests.
- **Database Logging:** All attacks are permanently recorded in a MySQL database with timestamps.
- **Live Web Dashboard:** A clean, responsive UI built with EJS and CSS that auto-refreshes to show the latest network threats.

## 📥 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/arp-ids-project.git
cd arp-ids-project
```

### 2. Database Setup

Open your MySQL client and run the following SQL commands:

```sql
CREATE DATABASE arp_ids;
USE arp_ids;
CREATE TABLE alerts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    attacker_mac VARCHAR(17) NOT NULL,
    target_ip VARCHAR(15) NOT NULL,
    alert_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### 3. Backend Setup

Navigate to the `backend` directory and install Node.js modules:

```bash
cd backend
npm install
```

### 4. Python Agent Setup

Install the required Python library:

```bash
pip install requests
```

## 🏃‍♂️ Running the Project

**Step 1:** Start your MySQL service (or XAMPP).

**Step 2:** Start the backend server:

```bash
cd backend
node server.js
```

> ✅ Console should log: `Connected to SQL Database successfully!`

**Step 3:** Open the dashboard in your browser:

```
http://localhost:3000
```

**Step 4:** Start the Python IDS Agent in a new terminal:

```bash
python arp_ids.py
```

## 🧪 Testing the IDS

To simulate an ARP spoofing attack using Kali Linux on the same network:

```bash
sudo arpspoof -i [interface] -t [Target_IP] [Gateway_IP]
```

> ⚠️ The dashboard will instantly update and display the detected spoofed MAC address!
